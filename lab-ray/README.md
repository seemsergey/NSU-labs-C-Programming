# Вход
### строка 1
	целое число N - количество вершин внешнего многоугольника

### от строки 2 до N + 1
    2 целых числа - координаты этих вершин

### строка N + 2
    целое число M - количество дыр в многоугольнике

### далее M блоков по K + 1 строк каждый:
    первая строка блока: целое число K - количество точек в текущей дыре
    следующие K строк: 2 целых числа - координаты вершин этой дыры

### после ввода дыр:
    произвольное количество строк: 2 целых числа - координаты проверяемых точек

# Выход
	если введено неверное количество вершин многоугольника или вершин дыр, не указано
    или некорректно введено количество дыр, то "bad input"

    иначе если точка находится внутри многоугольника (или на его границе) и вне 
    всех дыр, то выводим "IN", если точка находится вне многоугольника или в дыре 
    (или на ее границе), то выводим "OUT"

# Ограничения
    По времени - 3 сек., по памяти - 3MB


## Краткое описание рекомендуемого для решения алгоритма:

### Ray casting

Проводим луч направленный из точки в любом направлении. Считаем количество пересечений с ребрами многоугольника. Если число четно - точка вне многоугольника, если нечетно - внутри.

Более подробное описание алгоритма: https://www.youtube.com/watch?v=RSXM9bgqxJM
