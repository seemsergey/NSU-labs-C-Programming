# Компоненты сильной связности (Алгоритм Косорайо)

Реализовать поиск компонент сильной связности графа

## Вxод
*   Строка 1:

    > Целое число *N* от 0 до 2000 - количество вершин.

*   Строка 2:

    > Целое число *M* от 0 до *N* * (*N* - 1) / 2 - количество рёбер.

*   Начиная со строки 3 и до строки *M* + 2:

    > Целое число от 1 до *N* - начало и конец ребра.

## Выxод

*   Если *N* не от 0 и до 2000, тогда выведите `Bad number of vertices`

*   Если *M* не от 0 и до *N* * (*N* + 1) / 2 или есть два повторяющихся ребра, тогда выведите `Bad number of edges`

*   Если номер вершины не от 1 to *N*, тогда выведите `Bad vertex`

*   Если количество строк меньше чем **M** + 2, тогда выведите `Bad number of lines`

*   Иначе выведите количество компонент сильной связности графа и для каждой компоненты сильной связности графа выведите все вершины в порядке возрастания.

## Ограничения

Время работы 1 секунда, исполняемый файл + данные не более `sizeof(void*)` + `sizeof(void*)` * *M* + `1024Kb`

# Пример 1

## Вxод
<pre>
    3
    2
    1 2
    2 3
</pre>

## Выxод
<pre>
    3
    1
    2
    3
</pre>

</pre>
