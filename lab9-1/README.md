# Вход
### строка 1
    два целых числа M и N через пробел -- количество строк и столбцов сетки, 1 ≤ M, N ≤ 1000
### следующие М строк
    каждая строка содержит ровно N целых чисел, разделенных пробелами -- 1 клетка проходима, -1 клетка непроходима (стена)
### строка M+1
    два целых числа -- координаты начальной точки (строка и столбец, нумерация с 0)
### cтрока M+2
    два целых числа -- координаты конечной точки (строка и столбец, нумерация с 0) 

# Выход
    если M или N не удовлетворяют условию 1 ≤ M, N ≤ 1000, то “Invalid grid size”
    если присутствуют нечисловые символы, неверное количество элементов в строках сетки или отсутствуют необходимые строки с координатами, то “Invalid data format”
    если начальная или конечная точка выходит за границы сетки, то “Invalid coordinates” 
    если начальная точка является стеной (-1), то “Start is wall”
    если конечная точка является стеной (-1), то “No path found” 
    если путь между точками отсутствует, то “No path found”
### строка 1
    длина кратчайшего пути (количество шагов между клетками)

# Ограничения
Время на работу программы не более 3 секунд, исполняемый файл + данные не более 16 * M * N + 10Mb
