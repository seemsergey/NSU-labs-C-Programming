# Вход
### строка 1
	целое N от 1 до 2000000 -- длина массива
### строка 2
	N целых чисел -- значения элементов массива

# Выход
    Если N не от 1 до 2000000 - bad input
    Если N<=10:  цепочка присваиваний, которая сортирует конкретный массив в формате: a && b => a_1 b_1 || ... || a_1 b_1 ... (В конце значения элементов массива упорядоченные по возрастанию с помощью алгоритма Merge Sort)
	Если N>10: значения элементов массива упорядоченные по возрастанию с помощью алгоритма Merge Sort, разделенные пробелами.

# Ограничения
Время на сортировку (без учета ввода-вывода) 3 секунды, исполняемый файл + данные не более 4 * N + 8Mb

# Пример 1
### Вход
<pre>
3
4 -1 2
</pre>
### Выход
<pre>
4 && -1 => -1 4 | -1 4 && 2 => -1 2 4 | -1 2 4
</pre>
