# Вход
### строка 1
	целое N от 0 до 50 -- количество операторов 
### от строки 2 до N+1
	через пробел символ оператора, тип операции (ADD, SUB, MUL, DIV), число M от 1 до 20 -- приоритет 
### строка N+2
	строка длиной не более 1000 символов -- арифметическое выражение
# Выход
	если на входе не арифметическое выражение, состоящее из целых чисел, скобок и заданных бинарных операций, то "syntax error";
	иначе если на входе арифметическое выражение и его вычисление приводит к делению на 0, то "division by zero";
	иначе значение арифметического выражения

# Ограничения
Исполняемый файл + данные не более 4Mb
